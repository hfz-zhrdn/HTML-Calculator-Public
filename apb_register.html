APB Register Configuration
This tool allows you to configure the APB registers for the selected
device. The parameters you enter will be used to calculate the output
values for the PLL registers. The output values can be used to
configure the PLL settings in your design.

Instructions:
1. Select your device from the Device dropdown above.
2. Enter the required parameters in the input fields (Can be retrieved from **).
3. Click "Calculate" to compute the output values.
